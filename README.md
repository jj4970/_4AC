# _4AC

лёгкий античит для анархических серверов

---

## описание

_4ac создан для серверов в стиле 2b2t.
он не ограничивает геймплей - только останавливает то, что ломает сервер.

---

## использование

закинь на сервер и настрой при необходимости

---

## примечание

это экспериментальный софт.
используй на свой риск.
